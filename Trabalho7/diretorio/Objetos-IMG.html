Objetos imagem

Existem diversos formatos de representação de imagem digital que podem ser utilizados em páginas HTML. Os formatos mais comuns são JPEG, PNG e GIF.

O formato JPEG ( Joint Photographics Expert Group ) faz uso de métodos de compressão com pouca perda.

O formato PNG ( Portable Network Graphics ) suporta métodos de compressão sem perda.

O formato GIF ( Graphics Interchange Format ) suporta diversos método de compressão sem perda com patente, o que pode restringir seu uso.
